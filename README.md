# Hyejin Portfolio

개발자로써 프로젝트를 돌아보고, 나를 알리기 위한 프로젝트 입니다.

## 접근 URI
http://hyejin-hwang.com

## 사용 기술

- Frontend
  - Javascript(ES6) / Html / styled-components
  - React
- DevOps
  - AWS(EC2)
  - Git, GitHub
  
## 실행/중지 명령어

```bash
# 프로세스 실행 스크립트
$ npm start

# 프로세스 삭제 명령어
$ pm2 delete [process id]

# 프로세스 확인 명령
$ pm2 ls
```
